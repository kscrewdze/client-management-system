## v9.1.0 — 2026-03-27

**Скачать:** [ClientManager-v9.1.0-win64.zip](https://github.com/kscrewdze/client-management-system/releases/tag/v9.1.0)

### Что нового в 9.1

- Вкладка «☕ Поддержка» — донат через DonationAlerts, ⭐ GitHub, 🐛 Issues
- Фильтр клиентов по статусу (Все / ✅ Выполненные / ⏳ В ожидании)
- Столбец «Дата завершения» в таблице и Excel-экспорте
- Release Manager — автоматическая сборка EXE, версионирование, упаковка
- GitHub Actions CI/CD — автосборка + публикация при push тега

### Оптимизации в 9.1

- Lazy-загрузка вкладок (быстрый запуск)
- PyInstaller `--onedir` вместо `--onefile`
- Исключены ~35 неиспользуемых Qt-модулей

### Установка (Windows)

1. Скачайте ZIP-архив из ссылки выше
2. Распакуйте в любую папку
3. Запустите `ClientManager.exe`

> База данных (`clients.db`) и экспортированные файлы сохраняются рядом с EXE.

## Как создать новый релиз

```bash
# Автоматически: bump версии + сборка + упаковка
python release_manager.py release patch    # 9.1.0 → 9.1.1
python release_manager.py release minor    # 9.1.0 → 9.2.0

# Публикация на GitHub (Actions создаст Release автоматически)
git add -A
git commit -m "release: v9.1.1"
git tag v9.1.1
git push origin master --tags
```
