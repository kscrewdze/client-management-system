# Changelog

Все заметные изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [9.2.0] — 2026-03-27

### Добавлено (9.2.0)

- **Установщик Windows (Inno Setup)** — `ClientManager-setup.exe` с выбором папки и ярлыком на рабочем столе
- **Поиск по всей базе** — теперь ищет по имени, телефону, Telegram, цене, дате, матрице, услуге, комментарию, числу судьбы (11 полей)
- **Тёмная тема** — Material Dark стиль (#121212, фиолетовые акценты)
- **Стеклянная тема** — полупрозрачное окно (opacity 0.92) с индиго/бирюзовыми акцентами
- **Кастомный заголовок окна** — безрамочное окно с собственной панелью заголовка, перетаскивание, кнопки ─ □ ✕
- **Кнопка «Предложить тему»** — ссылка на Telegram для предложения новых тем
- **Карточка «Написать в Telegram»** на вкладке поддержки — ссылка t.me/kscrewdze
- **Выделение карточек** — Donate (золотая рамка) и Telegram (синяя рамка) с увеличенными элементами
- **Альтернативная горячая клавиша** — Ctrl+` для консоли разработчика (для ноутбуков без F12)

### Исправлено (9.2.0)

- **Уведомления не прятались** — переписана логика анимации fade-out, гарантированный вызов hide()
- **Слои вкладок накладывались** — добавлен фон для QStackedWidget, вкладки больше не просвечивают
- **Кастомные матрицы показывались как «—»** — добавлен fallback на service_name
- **HTML-инъекция в debug-консоли** — SQL-вывод теперь экранируется через _esc()
- **Цены теряли копейки** — условное форматирование: целое или с 2 знаками
- **Emoji отображались как квадрат** — убраны из заголовка и кнопок, заменены на текст
- **Debug-консоль: безопасность** — whitelist (только SELECT), вместо blacklist

### Изменено (9.2.0)

- **Компактная панель клиентов** — кнопки + фильтр + поиск в одну строку
- **Тема-карточки** — горизонтальный компактный дизайн (68px), 2 колонки
- Заменены темы: Рассветная → Тёмная, Полуночная → Стеклянная
- Обновлены зависимости: PySide6 6.11.0, Pillow 12.1.0, openpyxl 3.1.5, python-dateutil 2.9.0
- Версия заголовка перенесена в статус-бар

## [9.1.0] — 2026-03-27

### Добавлено (9.1)

- Вкладка «☕ Поддержка» с донатом (DonationAlerts), ⭐ GitHub, 🐛 Issues
- Фильтр клиентов по статусу выполнения (Все / ✅ Выполненные / ⏳ В ожидании)
- Столбец «Дата завершения» в таблице клиентов и Excel-экспорте
- Release Manager CLI (`release_manager.py`) — автосборка EXE, версионирование, упаковка
- GitHub Actions CI/CD — автоматическая сборка и публикация релиза при push тега `v*`
- `version.py` — единый источник версии приложения
- Файл `RELEASES.md` — история релизов для GitHub

### Оптимизировано

- Lazy-загрузка вкладок (ускорение запуска — только Statistics + Clients загружаются сразу)
- Отложенный авто-экспорт при старте приложения
- PyInstaller `--onedir` вместо `--onefile` (быстрый холодный старт)
- Исключены ~35 неиспользуемых PySide6/Qt-модулей из сборки
- Сборка (dist, build) вынесена в `output/` — не засоряет проект
- Архив EXE по версиям: `output/exe_archive/ClientManager-v{X.Y.Z}/`

## [9.0.0] — 2026-03-26

### Добавлено (9.0)

- Полная миграция GUI на **PySide6 (Qt6)** с CustomTkinter
- Анимированный **Splash Screen** с прогресс-баром при загрузке
- **6 премиум-тем** оформления: Изумрудная, Сапфировая, Рубиновая, Аметистовая, Полуночная, Рассветная
- QSS-движок тем с мгновенным переключением без перезагрузки
- **Вкладка «База данных»** — встроенный SQL-браузер с просмотром таблиц
- **Вкладка «Темы»** — карточки тем с предпросмотром
- 8 горячих клавиш (Ctrl+1..7, Ctrl+R, Ctrl+E, Ctrl+D, Ctrl+F, Ctrl+Enter, Ctrl+Q, F12)
- **Статистика** — дашборд с 6 метриками (клиенты, заработок, средний чек, выполнено, матрицы)
- Unicode-совместимый поиск (кириллица) через кастомную SQL-функцию `UNICODE_LOWER()`
- Автоматический расчёт **Числа Судьбы** из даты рождения
- Автозаполнение названия и цены при выборе матрицы
- Потокобезопасная база данных с `threading.Lock`
- Экспорт клиентов и матриц в Excel (`.xlsx`)
- Параметризованные SQL-запросы (защита от инъекций)
- HTML-экранирование в экспорте (защита от XSS)
- Поддержка PyInstaller для сборки standalone `.exe`
- Поддержка `sys._MEIPASS` для корректной работы в frozen-режиме
- Логирование через `logging` вместо `print()`

### Удалено

- Зависимость от CustomTkinter
- Зависимость от pandas (заменён на openpyxl)
- Неиспользуемые модули: `widgets/`, `themes/manager.py`, `utils/clipboard.py`, `utils/shortcuts.py`, `utils/timer_manager.py`

### Исправлено

- `BaseTheme.__init__()` больше не перезаписывает имя дочерней темы
- Кнопка «Обновить» больше не передаёт `False` в `refresh()`
- Тултипы корректно отображаются на всех столбцах таблицы
- Выделение строк в таблице клиентов — текст не сливается с фоном
- Конфиг темы сохраняется в `APP_DIR`, а не во временную папку `_MEIPASS`
